# HP India Product Image Links — Complete Guide for Jetage India

## How to Download Images from HP Store

1. Visit the product page URL below
2. Right-click the main product image → **"Open image in new tab"**
3. Save the image (usually PNG, from `ssl-product-images.www8-hp.com` or `in-media.apjonlinecdn.com`)
4. Convert to WebP using [Squoosh](https://squoosh.app/) (quality 75, 800x600 max)
5. Rename to `front.webp` and place in the correct folder

## HP Image CDN URL Patterns

When you open an image from the HP store, it will come from one of these domains:

### Primary: `ssl-product-images.www8-hp.com`
```
https://ssl-product-images.www8-hp.com/digmedialib/prodimg/lowres/{CODE}.png
```

Examples that work:
- `https://ssl-product-images.www8-hp.com/digmedialib/prodimg/lowres/c07989500.png` ✅ (Pavilion 15)
- `https://ssl-product-images.www8-hp.com/digmedialib/prodimg/lowres/c06335594.png` ✅ (Color Laser 150nw)

Examples that DON'T work (from products.ts):
- `https://ssl-product-images.www8-hp.com/digmedialib/prodimg/lowres/c08107811.png` ❌ 404

**The `c081078XX` series URLs are fabricated and do not exist.** You must find the correct image code from the actual HP store pages.

### Alternative: `in-media.apjonlinecdn.com`
```
https://in-media.apjonlinecdn.com/catalog/product/cache/b3b166914d87ce343d4dc5ec5117b502/{SKU}/{IMAGE}.png
```

Examples:
- `https://in-media.apjonlinecdn.com/catalog/product/cache/b3b166914d87ce343d4dc5ec5117b502/A/5/A58WCA-1_T1744185956.png` (Laser 303d)
- `https://in-media.apjonlinecdn.com/catalog/product/cache/b3b166914d87ce343d4dc5ec5117b502/h/p/hp_laser_303dw_printer_xian_sf_wireless_jetblack_catalog_whitebg_front_sup_m2957228.png` (Laser 303dw)
- `https://in-media.apjonlinecdn.com/catalog/product/cache/b3b166914d87ce343d4dc5ec5117b502/4/9/499N4A-1_T1720152495.png` (Color LaserJet Pro 3203dw)

## How to Update `products.ts`

Change each product's image paths from:
```typescript
image: "https://ssl-product-images.www8-hp.com/digmedialib/prodimg/lowres/c08107811.png",
images: [
  "https://ssl-product-images.www8-hp.com/digmedialib/prodimg/lowres/c08107811.png",
  "https://ssl-product-images.www8-hp.com/digmedialib/prodimg/lowres/c08107812.png",
],
```

To:
```typescript
image: "/images/products/printers/hp-laser-303d/thumbnail.webp",
images: [
  "/images/products/printers/hp-laser-303d/front.webp",
  "/images/products/printers/hp-laser-303d/side.webp",
],
```

## Quick Download Steps (Per Product)

1. Open the HP India store URL in Chrome
2. Right-click the large product image on the page
3. Select **"Open image in new tab"**
4. Save the image (usually 800-1200px PNG)
5. Go to https://squoosh.app/
6. Drop the PNG, convert to WebP, quality 75, resize to 800px max width
7. Download as `front.webp`
8. Place in `public/images/products/{category}/{product-id}/`
9. Update `products.ts` with the new local path
